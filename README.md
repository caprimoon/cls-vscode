# 财联社电报查看器

这是一个VSCode插件，用于查看财联社电报内容。

## 功能

- 在VSCode左侧活动栏中添加财联社电报图标
- 点击图标显示侧边栏视图，展示电报列表
- 自动刷新功能，可设置刷新间隔（秒）
- 新电报通知功能，显示电报具体内容
- 点击列表中的电报项查看详情
- 标记新电报，已读后自动去除标记
- 查看详情时打开的markdown文件不会提示保存
- 查看全部电报功能，一次性浏览所有电报

## 使用方法

### 查看电报列表

1. 点击VSCode左侧活动栏中的财联社电报图标
2. 侧边栏将显示电报列表

### 获取最新电报

1. 在侧边栏中，点击刷新按钮获取最新电报
2. 如果有新电报，将收到通知提醒，通知中包含电报的具体内容

### 查看电报详情

1. 在侧边栏的电报列表中，点击任意电报项
2. 电报详情将在新编辑器标签页中打开
3. 查看后，新电报标记将自动去除
4. 关闭详情页面时不会提示保存

### 查看全部电报

1. 在侧边栏中，点击"查看全部电报"按钮
2. 所有电报将在新编辑器标签页中打开，按顺序排列
3. 关闭详情页面时不会提示保存

### 清除新电报标记

1. 在侧边栏中，点击"清除通知"按钮
2. 所有新电报标记将被清除

### 自动刷新设置

1. 打开VSCode设置（Ctrl+, 或 Cmd+,）
2. 搜索"财联社电报查看器"
3. 配置以下选项：
   - 启用自动刷新：勾选此项以启用自动刷新功能
   - 自动刷新间隔：设置自动刷新的时间间隔（秒），默认为10秒
   - 启用新电报通知：勾选此项以启用新电报通知
   - 最大保存电报数量：设置保存的电报数量上限

注意：切换自动刷新功能已移至设置中，不再显示在侧边栏按钮中。

## 开发

- 安装依赖：`npm install`
- 运行插件：按F5启动扩展开发主机
- 打包插件：`vsce package`
